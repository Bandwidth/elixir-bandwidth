    Create an outbound phone call.

    Example: 

    case Bandwidth.Resources.Calls.create(client, %{ to: "+12223334444", from: "+19199993333" }) do
  {:ok, {201, _, _}} -> IO.puts "Call created"
  {:error, reason}   -> IO.puts "Error: #{reason}"
end
